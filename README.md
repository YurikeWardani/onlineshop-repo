# onlineshop-repo
onlineshop-repo adalah contoh projek website yang bergerak pada bidang jual-beli secara online.

# Dibangun dengan
- Framework Code Igniter
- Bootstrap
- HTML
- JavaScript
- PHP
- CSS
- MySQL database
